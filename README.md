# ezagent42

Claude Code 插件市场。

## 安装

```bash
# 添加市场
/plugin marketplace add ezagent42/ezagent42

# 安装插件
/plugin install dev-loop-skills@ezagent42
```

## 插件列表

| 插件 | 说明 |
|------|------|
| **dev-loop-skills** | AI 驱动的开发闭环 — 7 个 skill 覆盖 bootstrap、测试计划、测试编写、测试执行、功能评估、artifact 管理 |
| **prd2impl** | PRD-to-Implementation 流水线 — 14 个 skill 覆盖 PRD 结构化、gap 扫描、任务生成、批次调度、并行派发、里程碑验证 |
| **zchat** | IRC Channel — 通过 slash command 收发 IRC 消息 |
| **feishu** | 飞书 Channel — 通过 WSClient 长连接收发飞书消息 |
